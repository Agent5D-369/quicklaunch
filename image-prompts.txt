CANVA SETUP INSTRUCTIONS
================================================================

1. Open Canva → Create new design
2. Set custom dimensions as specified per image above
3. Go to Elements → Magic Media → Text to Image
4. Paste the prompt (remove the NOTE lines)
5. Select style: "Photo" for photorealistic prompts,
   "Illustration" for conceptual/3D prompts
6. Generate 4 variations, download the best at highest resolution
7. Save to: assets/images/[filename as specified above]

ADDITIONAL TIPS:
- For the book cover: generate the base image, then add text
  in Canva using the brand fonts (Montserrat 900 for title)
- For Rick's portrait: use the placeholder until a real photo
  is available — do not publish the AI placeholder on live site
- For OG image: add text overlay in Canva after generating
  the background — use Montserrat 900, white, 72px+
- Run all final images through Canva's "Background Remover"
  if you need transparent versions (for badges, logos, etc.)
- Export all photos as JPG at 85% quality for web performance
- Export any transparent elements as PNG
