# QuickLaunch Measurement Setup

## GA4
- Add your measurement ID to `assets/js/site-config.js`
- Field: `ga4MeasurementId`
- Example: `G-XXXXXXXXXX`

## Microsoft Clarity
- Add your Clarity project ID to `assets/js/site-config.js`
- Field: `clarityProjectId`
